# 🚀 VisionChat AI

A full-stack AI application that allows users to:

- Upload images 🖼️
- Generate captions 🧠
- Ask questions about images 💬
- Store chat history 📂 (MongoDB)

---

## 🛠 Tech Stack

Frontend:
- React.js
- CSS

Backend:
- FastAPI
- Transformers (BLIP model)

Database:
- MongoDB

---

## ⚡ Features

- Image Captioning
- Visual Question Answering (VQA)
- Chat history (session-based)
- ChatGPT-like UI
- Responsive design
